# How-to guides

*Task-oriented. Each guide answers a single "how do I…?" for someone who already
knows the basics. Recipes, not lessons.*

## Available

- **[Write chord progressions](write-chord-progressions.md)** — colon tokens,
  recipes, inversions, pedals.
- **[Build percussion patterns](build-percussion-patterns.md)** — letters,
  durations, modifiers, interrupters.
- **[Write melodies](write-melodies.md)** — scale degrees, anchoring, transforms.
- **[Create an arrangement](create-an-arrangement.md)** — YAML song files, sections.
- **[Import a lead sheet](import-a-lead-sheet.md)** — turn a chart PDF into a
  song.yml (chords + form; an agent workflow + a tested chord-symbol mapper).
- **[Render audio](render-audio.md)** — FluidSynth + ffmpeg via `render.py` /
  `play_music`.
- **[Use the web instrument](use-the-web-instrument.md)** — make music in the browser:
  the tabs, what each knob does, saving & exporting, troubleshooting (plus how to run it locally).
- **[ChordBuilder quick start](use-chordbuilder.md)** — the tap-based progression
  builder: build, audition, save, and carry a progression into the full instrument.
- **[Set up for development](set-up-for-development.md)** — venv, `make` targets,
  testing, and linting.
