Music Generator

A compact token language describes chords, percussion, and melodies. The engine realizes them into harmony, voices, and percussion, writes a MIDI file, and optionally renders audio through FluidSynth.

Try it live View on GitHub Hear examples

The token DSL

The core of the project is a small mini-language. A chord is a colon token, root[:inversion][:recipe][/bass]; percussion and melody have their own grammars. A whole progression is just a sequence of tokens.

Chords

C::maj7      C major seventh
A:1:min9     A minor ninth, 1st inv.
G::13        G dominant thirteenth
G::maj/C     G major over a C pedal

Melody

q1 q5 e4 e3 e2 e1 q7
# scale degrees with
# durations; transform
# via invert / transpose

Percussion

qk,eh,esh,eh
# kicks, hats, snares as
# duration+instrument
# tokens, with fills

Full reference: docs/token-grammar.md.

Examples

Played in your browser from the generated MIDI. Press play on any one.

Jazz progression

sevenths · ninths

An electric-piano ostinato over a ii–V coloured progression.

--mode ostinato --keys 'C::maj7, A::min9, D::min7, G::13' \
  --chords sevenths ninths --instrument epiano

Dense colour voicing

every chord tone

Full extended and exotic sets sounded complete across the register.

--voicing dense --instrument strings --chord-length w \
  --keys 'C::maj9, A::min11, F::maj7#11, G::13, E::mystic, Db::messiaen_resonance'

Process music

additive, after Glass

A single cell grows a note at a time, then contracts.

--process additive --process-cell 'e1 e2 e3 e5 e7 e5 e3 e2' \
  --instrument organ --melody-key C

Run it yourself

git clone https://github.com/galenspikes/music-generator
cd music-generator
python3 -m venv venv && venv/bin/pip install -r requirements.txt

venv/bin/python music_generator.py --mode ostinato \
  --keys 'C::maj7, A::min9, D::min7, G::13' --out my_song --no-play

Audio rendering additionally needs fluidsynth and ffmpeg. See the README for the full command surface.
